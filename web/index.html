What this project is

This repository contains a fully open hardware and software implementation of a handheld NDVI measurement device.

  • Exposes how NDVI measurements are produced
  • Supports validation, comparison, and experimentation
  • Acts as a reusable building block for larger sensing systems

Why it exists

Many vegetation indices are treated as black boxes, which makes validation, comparison, and integration difficult.

This project provides a transparent reference implementation with a known, inspectable signal path.

What NDVI means

NDVI compares reflected red light and near-infrared light to estimate vegetation health and density.

  • High NDVI: healthy, dense vegetation
  • Moderate NDVI: sparse or stressed vegetation
  • Low or negative NDVI: soil, hard surfaces, or water

Quick start

  1. Order components and fabricate the carrier PCB.
  2. Assemble the device using the wiring and assembly guides.
  3. Program the firmware onto the ESP32 Feather.
  4. Power the unit and begin taking NDVI measurements.
