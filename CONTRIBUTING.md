# Contributing to Open NDVI Handheld

Thank you for your interest in contributing to the Open NDVI Handheld project! This project aims to provide a transparent, accessible reference design for NDVI measurement instruments.

## How to Contribute

### Reporting Issues

- Use GitHub Issues to report bugs, suggest features, or ask questions
- Check existing issues before creating a new one
- Provide clear, detailed descriptions with steps to reproduce bugs
- Include relevant hardware/software versions and configurations

### Contributing Code

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-feature`)
3. Make your changes and test thoroughly
4. Commit with clear, descriptive messages
5. Push to your branch (`git push origin feature/amazing-feature`)
6. Open a Pull Request

### Code Guidelines

#### Firmware
- Follow Arduino coding conventions
- Comment extensively for educational value
- Test on actual hardware when possible
- Include calibration or validation data if relevant

#### Hardware
- Document design decisions in commit messages
- Include simulation results if available
- Provide rationale for component choices
- Update BOM and assembly documentation

#### Documentation
- Write clearly for diverse technical backgrounds
- Include practical examples and use cases
- Reference scientific sources for NDVI theory
- Test instructions on different skill levels

### Areas for Contribution

- **Firmware improvements**: Calibration algorithms, power management, connectivity
- **Hardware refinements**: Cost reduction, performance optimization, variant designs
- **Mechanical enhancements**: Ergonomics, durability, manufacturing improvements
- **Documentation**: Tutorials, theory explanations, troubleshooting guides
- **Analysis tools**: Data processing scripts, visualization tools, validation studies
- **Testing**: Field validation, comparison studies, reliability testing

## Community Standards

- Be respectful and inclusive
- Focus on educational and research value
- Share knowledge openly and transparently
- Acknowledge that this is not a commercial product
- Help others learn and understand NDVI measurement

## Getting Help

- Review existing documentation in the `docs/` directory
- Check the FAQ for common questions
- Ask questions in GitHub Issues
- Reference scientific literature for NDVI theory

Thank you for helping make NDVI measurement more accessible and transparent!
